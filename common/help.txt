__**TsukiBot Commands**__

**General Usage: ** `.tb <command> <parameters>`

__Commands__
`k`: Kraken
`g`: GDAX  
`p`: Poloniex
`c`: CryptoCompare in USD (multi)
`cs`: CryptoCompare in BTC (multi)
`cmc`: CoinMarketCap in USD (multi)
`cmcs`: CoinMarketCap in BTC (multi)
`b`: Bittrex (multi)
`n`: Binance (multi)
`e`: Etherscan (address or transaction)
`f`: Bitfinex
`kli`: KL Index value (multi)
`mc`: KL Comparison

__Personal Array__
`.tb pa`: Set your personal array
`.tb pa+`: Add a coin to your current pa
`.tb pa-`: Remove a coin from yout current pa

`.tbpa`: Call the array
`.tbpa+`: Call the array with BTC prices
`.tbpa*`: Call the array in expanded format
`.tbpa*`: Call the array ordered by % change 

__Extras__
`.tbstat`: Session Statistics

__Rooms and Roles__
`.tb join [valid coin]`: TsukiBot assigns the role to the user
`.tbleave`: Leave rooms
`.tblist`: Get the available rooms for the server

__(Owner Only)__
`.tb subrole [Name]`: Create expiring role
`.tb sub @User RoleName`: Grant a subscription

`.tb makeroom [valid coin]`: Create a private room and a role for a whitelisted coin
`.tbconfig`: Bot Permission Settings

`.tb shortcut [prefix]`: Custom prefix to access CMC in the server. Alphanumeric and/or `! $ % . _ , < > = + * &`; max. 3 characters.
(Use as `[prefix] eth btc elf`)


**Example usage**:
`.tb k dash usd` ⇒ Kraken price for DASHUSD
`.tb c btc bch zec` ⇒ CryptoCompare price for BTCUSD, BCHUSD, ZECUSD
`.tb p doge xmr` ⇒ Poloniex price for DOGEXMR
